# 量化技术文档

SGLang支持多种量化技术，通过权重量化、激活量化和KV缓存量化，在保持模型精度的同时显著减少内存使用和提升推理速度。

## 技术概览

量化系统支持多种量化方案：
- **权重量化**: INT8、INT4等权重精度降低
- **激活量化**: 激活值的动态或静态量化
- **KV缓存量化**: KV缓存的压缩存储
- **混合量化**: 不同层使用不同量化策略
- **校准算法**: 量化参数的优化校准

## 核心组件

### 量化方法
- **Post-Training Quantization**: 训练后量化
- **Quantization-Aware Training**: 量化感知训练
- **Dynamic Quantization**: 动态量化
- **Static Quantization**: 静态量化

### 量化精度
- **INT8**: 8位整数量化
- **INT4**: 4位整数量化
- **FP8**: 8位浮点量化
- **混合精度**: 多种精度的组合使用
